# Viscera Cleanup Detail (Archipelago world)

Work in progress. This repo will hold the Archipelago apworld for Viscera
Cleanup Detail, and any mod or tracker pieces it needs.

Start with `NEXT_APWORLD_PLAYBOOK.md`. It is the build playbook and process
guardrails distilled from the HP2PC project: what to stand up before writing
game logic, which architecture calls to get right early, and the mod-side
patterns worth reusing.
